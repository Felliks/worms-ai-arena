# Code of Conduct

This project expects practical, respectful collaboration.

## Expected Behavior

- Be direct about technical problems without personal attacks.
- Assume other contributors are working in good faith.
- Keep issue and PR comments focused on the code, design, tests, and user impact.
- Credit prior work when building on it.

## Unacceptable Behavior

- Harassment, threats, slurs, or targeted insults.
- Publishing private keys, private logs, personal data, or non-redistributable
  local files.
- Repeated off-topic arguments after maintainers ask to stop.

## Enforcement

Maintainers may edit or remove comments, close issues, reject pull requests, or
block contributors who violate this code of conduct. If you need to report a
problem, contact a maintainer privately or use GitHub's reporting tools.
